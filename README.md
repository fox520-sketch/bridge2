# 合約橋牌 GitHub Pages 部署檔

此壓縮檔只保留 `https://fox520-sketch.github.io/bridge/` 需要的靜態網站檔案。

## 上傳內容

- `index.html`
- `app.js`
- `style.css`
- `manifest.webmanifest`
- `service-worker.js`
- `cache-reset.html`
- `icons/`

Firebase rules 與設定範例請不要放在公開 GitHub Pages repo；請在 Firebase Console 另行部署與保存。


## v1.0.24.11 更新

- 主題選單新增「深色模式（OLED 黑底）」。
- 主題選單新增「黑白模式（電子紙）」，使用純黑白、低陰影設計，適合電子紙或需要高對比閱讀時使用。
- 自動主題在系統深色模式時改採 OLED 黑底深色模式。
